Projects

Todo App - React JS View Demo | View Code
This app was written in ReactJS using the new/experimental Hooks feature. I made this choice because Hooks makes the code easier to read by allowing for stateful functional components and a simplified state getting and setting process. Because state isnt only bound to classes and can be implemented in functional components, "this" closure is more predictable and function binding is skipped. I used local storage and the new lifecycle method useEffect() to store user data after the component updates. I am very proud of the results and use this app on my mobile often.

Landing Page w/ Multiple Themes - React JS View Demo | View Code
In this project I took a responsive HTML/SCSS webpage I coded, ported it to React, and added support for multiple themes. After transitioning the original HTML into React components, I extracted all reference to color in the CSS, compiled it into a state object, and passed it into the components as props. To make the code easier to read I defined my array of themes in a seperate file. A difficulty I had to overcome when making this project was making the SVGs transition along with the themes. Single color icons are able to transition by passing through a uniform fill property, though SVGs with more than one color needed to take multiple color props that were applied to their JSX as template literals in order to match each theme.
