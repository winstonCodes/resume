Projects

Todo App - React JS
I created this app in ReactJS using the new/experimental Hooks feature. Hooks made the state management easier to implement by simplifying state initiation and updating. Because state isnt only bound to classes and can be implemented in functional components, this closure is more predictable and function binding can be skipped. I used local storage and the new Hooks lifecycle method useEffect() to store user data. This was my first time creating a CRUD app in React without the guide of a course or tutorial and I am very proud of the results. One challenge I overcame was knowing when to save state to local storage

Landing Page w/ Multiple Themes - React JS
In this project I took a responsive HTML/SCSS webpage I coded, ported it to React, and added support for multiple themes. After transitioning the original HTML to multiple components, I extracted all reference to color in the CSS and made it into one shared state object theme. To make the code easier to read I defined an array of themes in a seperate file. A difficulty I had to overcome when making this change was how to make the SVGs transition along with the themes. SVGs of a single color were able to transition by passing through an inline fill property, though SVGs with more than one color were more complex. I made these into their own component.
